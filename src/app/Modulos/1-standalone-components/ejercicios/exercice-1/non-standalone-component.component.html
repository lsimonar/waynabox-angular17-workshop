Modulo 1: Standalone Components

Este componente es parte del Module1Module. Este modulo contiene varios componentes, pero solo uno se renderiza en este componente. El objectivo es refactorizar este componente para que sea un standalone component que solo importe el componente Dummy1.

Ejercicio 1: Refactorizar el componente NonStandaloneComponent para que sea un standalone component.

Ejercicio 2: Refactorizar el componente Dummy1 para que sea un standalone component sin modificar ningun otro componente.

Ejercicio 3: Quitar el export del componente Dummy1 en el Module1Module.
